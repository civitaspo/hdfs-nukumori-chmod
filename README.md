# Hdfs::Nukumori::Chmod

## Usage

```
$ bundle exec hdfs-nukumori-chmod -p /user/civitaspo -c hdfs-site.xml,core-site.xml
```

chmod 755 recursive, but chmod 644 if the object is file.
